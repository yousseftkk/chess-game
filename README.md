# Chess Game (Java) ♟️

## Overview
A fully playable chess game built in Java implementing standard chess rules and turn-based gameplay.

## Features
- Legal move validation
- Object-oriented design (pieces, board, game logic)
- Turn-based system
- Game state tracking

## How to Run
1. Clone the repository
2. Open in IntelliJ / Eclipse
3. Run Main.java

## Tech Stack
- Java
- Object-Oriented Programming
